コードブロックのテスト

文中に短いコードを埋め込むには、`const greeting = "Hello";` のように、バッククォート（`）で囲みます。

複数行のコードは、以下のように3つのバッククォートで囲み、言語名を指定すると、自動で色付けされてとても見やすくなります。

// C++のサンプルコード
#include <iostream>
#include <string>

int main() {
    std::string name = "World";
    std::cout << "Hello, " << name << "!" << std::endl;
    return 0;
}
アルゴリズム 平均計算量 最悪計算量
バブルソート O(n^2) O(n^2)
クイックソート O(n log n) O(n^2)
マージソート O(n log n) O (n log n)
